# Startup Funding Success Probability

## 📌 Project Overview
This project analyzes factors influencing the probability of startup funding success using Python and pandas.  
The goal is to identify key predictors such as team size, founder experience, industry type, and growth rate.

## 📊 Dataset Description
- Source: Synthetic startup dataset (Phase 1–2 cleaned and analyzed)  
- Columns: Startup Name, Industry, Founder Experience, Team Size, Growth Rate, Funding Status  
- Missing values handled and dataset standardized for analysis.

## 🔍 Statistical Methods
- Descriptive statistics (mean, median, std, quartiles, percentiles)  
- ANOVA tests for categorical variables (Industry, Founder Experience)  
- Probability calculations for funding success  
- Visualizations using matplotlib and seaborn.

## 📈 Results & Insights
- Larger team size and higher founder experience increase funding probability.  
- Certain industries (Tech, Healthcare) show higher success rates.  
- Growth rate strongly correlates with funding outcomes.

## 🖼️ Visual Insights
- Bar charts for industry vs. funding success  
- Boxplots for founder experience impact  
- Probability distribution plots for growth rate.

## 💡 Recommendations
- Investors should prioritize startups with experienced founders and scalable industries.  
- Growth metrics are critical indicators for funding decisions.  
- Balanced team size improves success probability.

## 🚀 Future Scope
- Extend dataset with real startup data.  
- Apply machine learning models for predictive analysis.  
- Explore cross‑country comparisons of funding success.
